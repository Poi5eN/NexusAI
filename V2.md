## The Core Engine Change — Agentic Loop

The single biggest upgrade is replacing your current** **`POST /chat → LLM → stream text` with a proper** ** **agentic tool-calling loop** . Here's the pattern:

```typescript
// server/src/core/agent-loop.ts
async function* agentLoop(persona: Persona, messages: Message[], tools: Tool[]) {
  while (true) {
    const response = await openrouter.chat({
      model: persona.model,
      messages,
      tools,
      tool_choice: "auto",
      stream: true
    });

    // Stream thinking tokens to client as they arrive
    yield { type: "token", content: response.delta };

    if (response.finish_reason === "tool_calls") {
      for (const toolCall of response.tool_calls) {
        yield { type: "tool_start", tool: toolCall.name, input: toolCall.arguments };
      
        const result = await executeTool(toolCall.name, toolCall.arguments);
      
        yield { type: "tool_result", tool: toolCall.name, result };
      
        // Append tool result back to messages and loop again
        messages.push({ role: "tool", content: result, tool_call_id: toolCall.id });
      }
      // Continue loop — LLM will now reason over tool results
    } else {
      yield { type: "done" };
      break; // LLM gave final answer
    }
  }
}
```

This one loop replaces everything. Each persona just provides a different** **`tools[]` array and system prompt.
